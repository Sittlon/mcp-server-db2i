# Reporting security issues

Please **do not** open a public GitHub issue for security problems.

Email the maintainer (see `package.json` → `author` / `bugs.url`) with:
- A description of the vulnerability
- A minimal reproduction (config snippet, exploit query, etc.)
- The version of `mcp-server-db2i` affected
- Your assessment of impact

We aim to acknowledge reports within 72 hours and to ship a fix or mitigation within 30 days for high-severity issues.

For the broader threat model and the layered defenses this server provides, see [`docs/SECURITY.md`](docs/SECURITY.md).
